## 1. Resumen teórico

**1. Definición de proposición**

Es un enunciado o sentencia declarativa de la cual se puede afirmar, de forma inequívoca, que es verdadera o falsa, pero no ambas a la vez. Las preguntas, órdenes o exclamaciones no se consideran proposiciones.

**2. Tipos de proposiciones**

- **Simples (Atómicas):** Son aquellas que expresan una sola idea y no tienen conectores lógicos (ej. "El cielo es azul").
- **Compuestas (Moleculares):** Son las que están formadas por dos o más proposiciones simples unidas mediante conectores lógicos (ej. "El cielo es azul y hace calor").

**4. Explicación de tablas de verdad**

Es un método gráfico que sirve para determinar el valor de verdad de una proposición compuesta bajo todas las combinaciones posibles de valores de sus componentes. Permite clasificar los resultados en Tautología (todo verdadero), Contradicción (todo falso) o Contingencia (mezcla de ambos).

## 4. Reflexión personal

**1. ¿Qué fue lo más difícil de entender?**

Lo más complejo fue el proceso de simplificación utilizando las leyes lógicas. Al principio, es difícil identificar qué ley aplicar cuando una expresión es muy extensa (como las leyes de Absorción o De Morgan), ya que un pequeño error en un signo o un paréntesis puede cambiar totalmente el resultado de la tabla de verdad. 

**2. ¿Qué tema comprendí mejor?**

La construcción de tablas de verdad y la identificación de conectores lógicos. Comprendí fácilmente cómo, a partir de los valores de verdad de proposiciones simples, se pueden evaluar estructuras complejas. Me resultó muy útil aprender que cada conector tiene una regla fija (como que la condicional solo es falsa si el antecedente es verdadero y el consecuente falso) para determinar si un argumento final es una tautología, contradicción o contingencia.

**3. ¿Cómo puedo aplicar la lógica en mi carrera?**

La lógica es el pilar fundamental de la Ingeniería en Computación, ya que permite estructurar el pensamiento analítico necesario para resolver problemas complejos. Sus aplicaciones principales son:

- Programación y Algoritmos: Es la base de las estructuras de control (if, while). Permite que el software tome decisiones precisas mediante la evaluación de condiciones lógicas.

- Optimización: El uso de leyes proposicionales (como De Morgan o Absorción) permite simplificar expresiones de código y diseños de circuitos, logrando sistemas más rápidos y eficientes.

-	Hardware: En la arquitectura de computadores, la lógica se traduce físicamente en compuertas (AND, OR, NOT), que son los componentes básicos para el procesamiento de datos.
